I am learning Emacs

2024/01/19

Introduction

This may sound like the most sadistic thing to do, but I am going to be full-committing to Emacs starting next week. My philosophy with studying has been going under changes as of recent, and I think it is time for me to actually bite the bullet and learn Emacs

Philosophy behind the switch

Now all of this isn’t to say that I think Emacs is the perfect tool for me, but Emacs is a tool that I’ve been wanting to learn for some time but haven’t given it a chance. It’s same sort of thing as when I was studying Korean. I wanted to learn Japanese but didn’t want to take the dive into it and went with something I thought that “objectively easier.” It is silly, but that’s what I was thinking. It’s also what I’ve been going through with Neovim, as I have been trying to make vimwiki and Zettelkasten work. I am not liking my longer load times, and think that trying to cram as much as I am into Neovim isn’t going to work long term.

Since I have been trying to make Neovim an everything tool, it’s become too bloated to my liking, and I think going forward as an academic tool, Emacs is going to be worthwhile. Neovim doesn’t have a front-end interface or a client like Emacs, which means everything I do with Neovim has to be done from the terminal. This means there’s a level of abstraction that I don’t want to deal with, since this leads to me being very distracted. So Emacs for me is going to be an educational tool only.

Emacs the tool

Emacs is a very powerful tool for learning. Org-mode for one is a tool that’s appealed to me for a long time, and Neovim doesn’t have a good alternative. It has evil-mode, which means I can continue using the default vim bindings. It’s going to be a struggle since I have to move my Neovim workflow over from Neovim to Emacs, but I don’t think this will be a long-term problem.

I am essentially making Emacs into what Neovim-Tmux does for me already, only without the terminal. This may sound silly, but the terminal is very distracting to me. This removes that.

Emacs is going to be my primary tool for academic reading and note-taking. You can use Emacs as a PDF-reader, with it integrate Zettelkasten. You can also write and edit LaTeX side by side without having to launch separate programs (very cool!).

In conclusion

I am using Neovim to do more things that I want it to do. I am lightening the load by moving a large subset of what I do in Neovim over to Emacs, and will expand upon it in the future. For now, this is just the update I wanted to release for this week.
